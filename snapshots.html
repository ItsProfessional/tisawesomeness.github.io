Minecraft Snapshot Reverse Engineering

This is where I post my findings from looking at the source code for every new Minecraft snapshot. You can usually expect a post within 1-2 days of the snapshot releasing.

Notes

I will not try to reverse engineer features that will change a lot, such as ore distribution. If you want me to cover a specific feature or find out how something works, DM me on Discord at @tis_awesomeness.

I will be adding posts for earlier snapshots until all of them are finished, so check back every once in a while for new posts.

How can I do this?

Download DecompilerMC and run python main.py -c -mcv snap -s server to decompile and map the latest snapshot. Then open the source code in your favorite editor (I recommend IntelliJ) and start looking around.
